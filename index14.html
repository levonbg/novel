Encoder-Decoder模型引入了Attention model
什么是soft Attention model, hard Attention model , static Attention model
Transformer 长程依赖的捕获能力是比 RNN 类结构差的。
CNN/RNN/Transformer(self-attention,增加rnn提升效果)

Attention 函数的本质可以被描述为一个查询（query）到一系列（键key-值value）对的映射，就是个softmax
在计算 Attention 时主要分为三步:
1.是将 query 和每个 key 进行相似度计算得到权重，常用的相似度函数有点积，拼接，感知机等
2.一般是使用一个 softmax 函数对这些权重进行归一化
3.将权重和相应的键值 value 进行加权求和得到最后的 Attention。
主要亮点
1.用 Attention 机制代替了 RNN 搭建了整个模型框架
2.提出了多头注意力（Multi-headed Attention）机制方法
由编码器和解码器组成
    编码器栈式搭建了 N 个块
        编码器的一个网络块中
            一个多头 Attention 子层
            一个前馈神经网络子层组成
    解码器栈式搭建了 N 个块
        解码器的一个网络块中
            解码器的一个网络块中多了一个多头 Attention 层
attention函数执行多次，就是多头attention了，结果拼接，再做一次线性变换
在并行方面，多头 Attention 和 CNN 一样不依赖于前一时刻的计算，可以很好的并行，优于 RNN。
由于 Self-Attention 是每个词和所有词都要计算 Attention，所以不管他们中间有多长距离，最大的路径长度也都只是 1。可以捕获长距离依赖关系。
